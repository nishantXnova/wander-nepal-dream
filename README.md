# GoNepal - Discover the Land of the Himalayas

## About

GoNepal is a platform designed to inspire and assist global tourists in exploring Nepal. Discover Everest, ancient temples, wildlife safaris, and rich Nepali culture.

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase (Backend & Auth)

## Development

```sh
# Install dependencies
npm i

# Start development server
npm run dev
```

## Deployment

Deploy via the publish button in your project settings.
